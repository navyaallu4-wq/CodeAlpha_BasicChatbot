# 🤖 CodeAlpha Smart Chatbot

## 📌 Project Overview
CodeAlpha Smart Chatbot is a rule-based chatbot developed using Python for the CodeAlpha Python Programming Internship.

The chatbot interacts with users through the console and provides useful features like greetings, quotes, jokes, facts, calculator, guessing game, mood checker, and chat history.

---

## 🚀 Features

- 👋 Greeting System
- 📅 Current Date
- ⏰ Current Time
- 💡 Motivational Quotes
- 📚 Interesting Facts
- 😂 Random Jokes
- 😊 Mood Checker
- 🧮 Calculator
- 🎮 Guess Number Game
- 💬 Chat History
- 📖 Help Menu
- ℹ️ About Section
- 📊 Session Summary

---

## 🛠️ Technologies Used

- Python 3
- datetime
- random
- time

---

## ▶️ How to Run

1. Open Terminal / PowerShell
2. Navigate to the project folder

```bash
cd CodeAlpha_BasicChatbot
```

3. Run

```bash
python chatbot.py
```
